# AgriYieldSimulator

Simulação de produtividade agrícola baseada em condições climáticas.

Este projeto utiliza dados de temperatura e precipitação para simular produtividade de culturas e gerar recomendações simples sobre períodos de plantio ou colheita utilizando Python e ferramentas de análise de dados.

## Status do Projeto

🚧 Projeto em desenvolvimento.

A estrutura inicial do repositório foi criada e o código principal está sendo implementado gradualmente.

## Tecnologias Utilizadas

* Python
* pandas
* NumPy
* Matplotlib
* Seaborn
* Requests
